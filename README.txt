Установить на ПК python версии 3.9 и выше. https://www.python.org/downloads/

В папку domain_owners поместить secret_key.json (OAuth 2.0 Client IDs) (в этой версии пока 1 достаточно)
urls-list - список ссылок, которые необходимо проиндексировать

Запуск через start.bat
Приложение висит 5-6 сек. Это модифицируется библиотека python, чтобы не ставить много модулей на пк через cmd вручную

Дальше открывается браузер. Логин пороль от аккаунта owner. Далее, далее, далее. (можно и по другому сделать, там есть service_account в google рядом с OAuth 2.0 Client IDs, генириется аккаунт-почта и можно добавлять эту почту в owner сайта. при этом способе не нужнго будет вводить логин и пароль, но пока работает только с OAuth 2.0 Client IDs. переписать под service_account не сложно)

Индексация запускается.

Если захотели закрыть приложение, то лучше завершать комбинацией ctrl + c. Вызывается прерывание и происходит дозапись ссылок, которые не проиндексированы обратно в файл.
